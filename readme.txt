概要：
ボードゲームであるオセロ(リバーシ)プログラムです

ビルド（コンパイル＆リンク）：
visual studio2017でOseroG.slnを開き，上部にあるソリューション構成を『Release』にした後
『ビルド』→『ソリューションのビルド』

実行：
Othello.exe

-----------------
プログラムの説明：
・オセロのプログラムです．
・メニュー画面では2人のプレイヤーの思考を決めることができます．
　・思考は現状『MANUAL』『RANDOM』『αβ』の3つがあります．
　　・『MANUAL』…手動．人間が操作したい場合．
　　・『RANDOM』…合法手からランダムに1つ選び打つ．
　　・『αβ』…αβ法という探索法に基づき最善手を打つ．これを選んだ場合，探索の深さを指定する．
　　　　　　　　3〜7があり，数字が大きいほど強い．
・両プレイヤーの思考を決め，『START』を押すと開始です．
・対戦画面の右側にはスコアとしてプレイヤーの石の数が，右上にはプレイヤーの手番が表示されます．
・思考が『MANUAL』以外だと，どんなに速く打つ場所が決まろうとも1500ms(1.5秒)待ちます．この時間はMydef.hのTIMELIMITで変更することができます．
・打つことができない状態(パス)になると相手の手番に自動的に移ります．
・両プレイヤーが打つことができなくなると終局です．結果が画面右上に表示されます．
・終局後はEnterで閉じます．
